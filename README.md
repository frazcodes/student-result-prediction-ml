# 🎓 Student Result Prediction (Machine Learning)

This project predicts whether a student will PASS or FAIL based on:

- Study Hours 📚
- Sleep Hours 😴
- Attendance 📊

## 🚀 Tech Used
- Python
- Pandas
- Scikit-learn

## ⚙️ Features
- Logistic Regression Model
- Data Scaling (StandardScaler)
- Accuracy Evaluation
- User Input Prediction

## 📌 How to Run

```bash
pip install -r requirements.txt
python main.py
💡 Example Input
Study Hours: 5
Sleep Hours: 7
Attendance: 80
✅ Output
Result: PASS
